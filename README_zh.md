# md-html

Markdown 目录编辑器，带侧边栏导航、实时预览和双向同步保存——在一个浏览器页面中浏览和编辑所有 Markdown 文件。

## 概述

md-html 是一个md文件可浏览器编辑，且可管理某个目录下所有html文件的skill。

md-html 实现了两个功能：

- **实时编辑与同步** — 将每个 `.md` 文件渲染为可编辑的自包含 HTML 页面，提供分栏编辑器和实时预览。按 Ctrl+S 可同时将编辑内容保存回 `.md` 和 `.html` 文件。
- **目录管理器** — 生成一个带侧边栏的页面，列出所有 HTML 文件（从 `.md` 渲染的和已有的独立 `.html`），支持点击切换导航。

结果：一个浏览器页面即可通过侧边栏在所有 Markdown 文档之间导航、内联编辑任意 Markdown 文档并实时预览，同时保持 `.md` 和 `.html` 文件同步——无需文件选择对话框，无需逐个打开文件。

## 功能特性

**编辑功能（每个文件）：**

- 分栏编辑器与实时 Markdown 预览
- 视图模式：分栏 / 只读 / 仅编辑器
- 双向同步保存：Ctrl+S 同时写入 `.md` 和 `.html`
- 同步服务器离线时的下载备用方案
- Markdown 渲染（markdown-it）、代码高亮（highlight.js）
- Mermaid 图表，支持点击放大
- 深色/浅色模式，自动生成目录
- Tab 键支持、滚动同步

**导航功能（管理器页面）：**

- 侧边栏目录树，可展开/折叠子目录
- 点击切换所有 HTML 文件
- 标记徽章：✎ 表示可同步（来自 `.md`），■ 表示独立文件（已有 `.html`）
- 侧边栏搜索过滤
- 键盘导航（方向键 + Enter）
- 当前文件高亮、面包屑导航
- 深色/浅色模式、响应式布局、文件计数和子目录徽章

## 安装

### 系统要求

- **Node.js** 12+（无外部依赖——仅使用内置模块）

### 安装步骤

克隆仓库，并复制到你的skill文件下即可使用：

```bash
git clone https://github.com/Lana-Lan/md-html.git
```
